# On the fly execute plugin

This plugin allows you to execute code in an isolated environment.

```zsh
plugins=(... fly-exec)
```



## Supported Environments

| Alias            | Description                                                                              |
| :--------------- | :--------------------------------------------------------------------------------------- |
| fly-exec-php     | Runs php code                                                                            |
| fly-exec-java    | Runs java code                                                                           |
| fly-exec-node    | Runs nodejs code                                                                         |
