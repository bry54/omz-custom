# Media Utilities plugin

This plugin allows you to run several Utilities on media files.

```zsh
plugins=(... media-utilities)
```

## Supported Utilities

| Alias            | Description                                                                              |
| :--------------- | :--------------------------------------------------------------------------------------- |
| yt-dl            | Download anything from youtube                                                           |
| trim-media       | Trims media files                                                                        |
