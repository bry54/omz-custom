# Workflow plugin

This plugin allows you to call commonly perfomed actions.

```zsh
plugins=(... workflow)
```



## Supported Calls

| Alias              | Description                                                                              |
| :------------------| :--------------------------------------------------------------------------------------- |
| composer <command> | Calls composer dependency manager by executing supplied command                          |
| aws <command>      | Calls aws-cli tool executing supplied command                                            |
